# Jiaying Cao Personal Skills Library

这是 Jiaying Cao 的个人专用 skills 仓库，用于沉淀科研工作中常用的 AI 工作流能力与工具集。

## Skills 用途对照（科研/PPT 视角）

1. `imagegen`  
用途：生成或编辑图片（封面图、示意图、概念图、背景图）。  
适用：科研汇报视觉增强、PPT 配图快速产出。  

2. `pdf`  
用途：读取、抽取、检查、生成 PDF 文档。  
适用：论文资料整理、文献内容提取、报告导出与排版检查。  

3. `screenshot`  
用途：系统级截图（全屏、窗口、区域）。  
适用：实验结果截图、软件界面截图、论文图表截取后放入 PPT。  

4. `figma`  
用途：连接 Figma 设计稿并提取设计上下文与素材。  
适用：有 Figma 设计文件时，做设计稿到实现的衔接。  

5. `figma-implement-design`  
用途：按严格流程将 Figma 节点 1:1 还原到代码实现。  
适用：UI/前端落地场景；对纯科研汇报属于可选技能。  

## 推荐使用优先级

面向“科研汇报/PPT 制作”的推荐顺序：
1. `imagegen`
2. `pdf`
3. `screenshot`

`figma` 与 `figma-implement-design` 保留为扩展能力（当你涉及 UI 设计或网页实现时再使用）。

## 仓库目标

- 支持科研汇报材料（尤其是 PPT）设计与产出
- 统一常用技能配置，便于复用与迭代
